## 讨论

有同学会想到Xcode Snippets功能，其实在编写这个插件的初期我也考虑过这个问题。经过思考和体验发现，在初始化一个类的时候，这个插件更专业更高效，代码块更适合一段代码的快速编写场景。

## 安装
1. 下载工程后，把`Products/`下的`QuicAPP.app`拖到`应用程序`目录下。
2. 勾选`系统偏好设置 -> 扩展 -> Xcode Source Editor -> Quic`扩展。
3. 重启Xcode即可。
